# Autodox - Identity Orchestration Platform

Enterprise-grade identity verification and document management platform.

## Overview

Autodox provides a comprehensive identity orchestration solution with:
- Multi-brand white-label portals
- Document verification workflows
- Custom domain management
- Enterprise authentication

## Technology Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL, Edge Functions, Auth)
- **Integrations**: Netlify (domains/SSL), Resend (email)

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

The application is deployed via the platform's publish functionality.
